## Hypothesis

A cognitive architecture should not generate its own experiences.

Experiences should originate from interaction with an external world.

## Implementation

Implemented a simulated world.

The world generates observations.

The brain consumes observations.

## Experiment

Run the embryo.

Verify observations originate from the world.

Verify the brain updates every tick.

## Observations

The architecture successfully separates the world from cognition.

The embryo can continuously receive observations without directly creating them.

## Conclusion

The separation between world and brain improves modularity and better represents developmental intelligence.

Future work should make the world dynamic.
