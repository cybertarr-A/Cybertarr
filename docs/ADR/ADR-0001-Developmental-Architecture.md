# ADR-0001 — Developmental Architecture

## Status

Accepted

## Context

Most modern AI systems begin as highly trained models.

Cybertarr instead investigates whether intelligence can emerge through staged development.

## Decision

Cybertarr will begin as a Digital Embryo and acquire capabilities over time through interaction and experimentation.

## Consequences

Advantages:

* Supports continual learning.
* Encourages modular growth.
* Simplifies experimental evaluation.
* Aligns with the project's developmental philosophy.

Disadvantages:

* Slower initial progress.
* Requires extensive experimentation.
* Higher architectural complexity.

## Update

Decision:

The embryo will never create observations.

All observations originate from an external world.

Reason:

This separates cognition from the environment and allows multiple worlds to be connected without modifying the brain.
