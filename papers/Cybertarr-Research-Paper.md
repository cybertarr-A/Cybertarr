# Cybertarr

## Working Title

**Cybertarr: A Developmental Cognitive Architecture Inspired by Biological Growth**

## Abstract

Cybertarr investigates whether intelligence can emerge from a developmental computational architecture rather than a fully trained neural network.

The project begins with a minimal Digital Embryo possessing only an internal state and a heartbeat. Higher cognitive abilities are introduced incrementally and experimentally. Each capability must emerge from simpler mechanisms or be justified as a fundamental architectural component.

This document will evolve into the project's primary research paper as new experiments are completed.

## Current Progress

Cybertarr now consists of five interacting components:

- World
- Observation
- Brain
- Internal State
- Heartbeat

This demonstrates the first closed developmental loop where an external environment continuously provides experiences to an otherwise knowledge-free computational organism.
